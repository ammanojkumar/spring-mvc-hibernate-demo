1. Java 7
2. Tomcat 
3. Derby database (download squirrel sql)
4. run the script.sql - create LOGIN_USER table. STUDENT table creates when app starts

This is spring MVC application which has,
1. Hibernate configuration
2. Spring security
3. Spring exception handling
4. Rest service with ModelAndView
5. Cache and property place holder
6. Log4j
